# Mini Ai Generator

## teach
Adds a new question-answer pair.

## quote
Sends a random quote from the `quotes.txt` file.

## ask
Asks the bot a question.

# Fun

## hack
Hacks the specified user (not really, hehe)  
If no user is specified, hacks the person who called the command.  
*command ported from $sudo-bot*

## ball
Just answers the question (yes/no)  
*command ported from $sudo-bot*

# Util

## ban
Bans the specified user (by ID)

## unban / pardon
Unbans the specified user (by ID)

## kick
Kicks the specified user (by ID)  
Basically, this is a ban, immediately followed by an unban  
Usually useful in private groups

# Info

## ver / info
Sends the bot's version, telegram lib and programming language.

## uptime
Sends the bot's uptime.
